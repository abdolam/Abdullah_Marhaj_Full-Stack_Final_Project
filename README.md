## תיאור כללי

EcoStore היא אפליקציית חנות אונליין המדמה מערכת מסחר אלקטרוני אמיתית.  
המערכת פותחה כפרויקט גמר במסגרת קורס Full‑Stack Developer ב‑HackerU, ומדגימה עבודה מלאה בצד הלקוח (Frontend) ובצד השרת (Backend), כולל אימות משתמשים, ניהול מוצרים, עגלת קניות והזמנות.

בהתאם לדרישות הקורס, תוך הקפדה על ארכיטקטורה נכונה, אבטחה, וניקיון קוד.

---

## מטרות הפרויקט

- הדגמת יכולת פיתוח מערכת Full‑Stack מלאה
- עבודה עם REST API
- שימוש ב‑JWT לאימות והרשאות
- הפרדה ברורה בין Client ל‑Server
- מימוש תהליכי E‑Commerce בסיסיים
- פיתוח קוד קריא, מודולרי וניתן להרחבה

## טכנולוגיות

### Frontend

- React (Vite)
- React Router
- Context API
- Axios
- Tailwind CSS

### Backend

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (Authentication)
- Joi (Validation)

## מבנה הפרויקט

### Client

- api – תקשורת עם השרת
- components – קומפוננטות UI
- pages – עמודי מערכת
- hooks – Hooks מותאמים
- context / providers – ניהול State גלובלי

### Server

- modules – cart / product / order / user / category
- controllers – לוגיקת בקשות
- services – לוגיקה עסקית
- models – סכמות MongoDB
- routes – הגדרת נתיבים
- middlewares – אימות, הרשאות, טיפול בשגיאות

---

## הוראות התקנה והרצה

### Backend

1. כניסה לתיקיית server
2. התקנת חבילות:  
   npm install
3. יצירת קובץ .env עם משתני סביבה
4. הרצת השרת:  
   npm run dev

### Frontend

1. כניסה לתיקיית client
2. התקנת חבילות:  
   npm install
3. הרצת האפליקציה:  
   npm run dev
