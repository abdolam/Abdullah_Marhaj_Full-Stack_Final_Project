# EcoStore App – פרויקט גמר Full-Stack (HackerU)

אפליקציית E-Commerce מלאה המדגימה חנות אונליין אמיתית, כולל צד לקוח (React) וצד שרת (Node/Express) עם מסד נתונים MongoDB.

## סקירה כללית

EcoStore היא חנות אונליין המדגימה תהליך רכישה מקצה לקצה: מוצרים, עגלה, הזמנות, אזור אישי ואזור ניהול (Admin).

## טכנולוגיות

Frontend: React, Vite, TailwindCSS  
Backend: Node.js, Express, MongoDB, JWT

## פיצ'רים עיקריים

- קטלוג מוצרים
- עגלת קניות והזמנות
- אזור אישי למשתמש
- Admin Panel לניהול
- איפוס סיסמה באמצעות מייל

## התקנה והרצה

### Backend

npm install  
npm run dev

### Frontend

npm install  
npm run dev
